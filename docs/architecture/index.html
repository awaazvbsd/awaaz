Data Flow Steps

Step 1: User Input

User records voice sample through browser interface. Audio is captured in WAV/MP3 format.

Step 2: Frontend Processing

React UI validates and prepares audio data for transmission via REST API call to Flask backend.

Step 3: Backend Analysis

Flask API receives audio and forwards to Praat-parselmouth for voice biomarker extraction.

Step 4: Biomarker Extraction

Praat analyzes voice and extracts scientific features: Pitch, Jitter, Shimmer, HNR, Formants, Intensity.

Step 5: AI Analysis

Biomarkers and context (questionnaire, live answers) sent to Google Gemini API for psychological interpretation.

Step 6: Recommendation Generation

Gemini generates personalized immediate actions, long-term strategies, and next session timing.

Step 7: Results Display

React UI presents comprehensive results to user with visual indicators and actionable suggestions.
